Bienvenue dans le jeu de Amine - Pac-Man

Instructions :

  • Appuyez sur D pour activer/désactiver le mode debug.
  • Appuyez sur W pour ajouter un véhicule errant.
  • Appuyez sur L pour activer le mode "file indienne".
  • Appuyez sur un endroit de l'écran pour ajouter un obstacle.
  • Collectez des étoiles pour augmenter votre score.
  • En mode "file indienne", évitez les obstacles sinon vous perdez des vies.
  • Si vos vies tombent à zéro, vous perdez et l'écran "Game Over" apparaît.
  • Utilisez le bouton Restart pour rejouer après un Game Over.
